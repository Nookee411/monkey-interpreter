# Monkey language interpreter

Monkey language is a simple declaration of language with basic operations.
It's conception is taken from book "Writing An Interpreter in Go" by Thorsten Ball

## Basic syntax

### Defining a variable

```
let age = 1;
let name = "Monkey";
let result = 10 \* (20 / 2);
```

### Supported variable types

Integers 1,2,3,4
Strings "a", "b", "c"
Arrays \[1,2,3,4\]
HashMaps { a: 1, b: 2, c:3 }

### Functions

Functions are defined with `fn` keyword

```
let add = fn(a, b) { return a + b; };
```

Also supports implicit return

```
“let add = fn(a, b) { a + b; };”
```

Calling a function is trivial

```
add(1, 2);
```

Also supports recursion

```
let fibonacci = fn(x) {
  if (x == 0) {
    0
  } else {
    if (x == 1) {
      1
    } else {
      fibonacci(x - 1) + fibonacci(x - 2);
    }
  }
};
```

Support for higher order functions also nessesary

```
let twice = fn(f, x) {
  return f(f(x));
};

let addTwo = fn(x) {
  return x + 2;
};

twice(addTwo, 2); // => 6
”
```

## Structural parts of interpreter

SourceCode -> Tokens -> AST -> Evaluation

### Lexical Analysis. SourceCode -> Tokens

let x = 5 + 5;
Transforms to:
[
LET,
IDENTIFIER("x"),
EQUAL_SIGN,
INTEGER(5),
PLUS_SIGN,
INTEGER(5),
SEMICOLON
]

## REPL Read, Eval, Print, Loop

Tokenizes input, Parses it to tokens, evaluates it, prints result

## Parses

Parser is the component that turns input into data structure that represents the input.
In out case parser will accept tokens, generated by lexer and produce AST.
We are going to use recursive descent parsing.
